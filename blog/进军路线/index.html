August 26, 2021 · One min read

0.1.9​

加入了分支的概念，你能想象么？

一个仓库可以被分支管理，切换出不同的分支互不影响，彼此独立，但是公用一套代码。

0.1.8​

加入了 dpChain 这个 api，链式触发 action。

好处就是大大的提升了跳转体验，可以直接跳转对应导出的代码位置，快速定位，相比 dp 好用了 一大截

0.1 版本前后​

主要区别就是，Redux 的生态更新的缘故，基本功能和 api 设计并没有发生太大变化，主要是扩展了功能
